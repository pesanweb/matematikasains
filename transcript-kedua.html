Kenapa PMF?

Karena sing_days itu diskrit — jumlah hari burung berkicau dalam sebulan. Nilainya cuma bisa bilangan bulat: 0, 1, 2, ..., 30. Nggak mungkin 12.5 hari. Makanya pakai PMF (Probability Mass Function), bukan PDF.

| | PDF (Gaussian, Uniform) | PMF (Binomial) |
|---|---|---|
| Jenis data | Kontinu (berat, panjang, rasio) | Diskrit (hari, jumlah, hitungan) |
| Titik spesifik | Peluang di satu titik = 0 | Peluang di satu titik ada nilainya |
| Fitur burung | wingspan, weight, beak_ratio | sing_days |

Hubungan dengan data burung:

sings_days = berapa hari burung berkicau dari 30 hari.

Ini cocok banget sama binomial karena:
  • n = 30 (jumlah percobaan tetap, 30 hari)

  • p = peluang sukses (probabilitas burung berkicau di satu hari)

  • Hasil cuma dua: berkicau atau nggak di tiap hari

  • Setiap hari independen (hari ini berkicau nggak pengaruh ke besok)


Rumusnya: 
P(k) = C(n,k) × p^k × (1-p)^(n-k)

  • C(n,k) = kombinasi "berapa cara milih k hari dari 30"

  • p^k = peluang berhasil k kali

  • (1-p)^(n-k) = peluang gagal sisanya


Kode paling gampang (pakai scipy):
from scipy.stats import binom
pmf = binom.pmf(val, n_trials, probability)


Simpel — cuma satu baris. binom.pmf udah ngitungin semua rumus kombinasi + pangkat-pangkatnya.

Tugas 2.1 — uniform_inverse_cdf (Rumus Ajaib Inverse CDF)

erfinv adalah fungsi matematika khusus. probability itu bilangan antara 0 dan 1.

Tugas Segmen 2.5 — Estimasi Parameter (Menebak miu, sigma, dll dari Data)



Kalau di dunia nyata, kita tidak tahu miu dan sigma sebenarnya. Kita harus menebak dari data yang ada. Ini caranya:

Gaussian — gaussian_params_estimation:

miu = np.mean(sample)
sigma = np.std(sample, ddof=1)  # ddof=1 untuk standar deviasi sampel


  • np.mean = rata-rata

  • np.std = standar deviasi (seberapa menyebar data)

  • ddof=1 = pakai rumus sampel (bukan populasi)


Binomial — binomial_params_estimation:

success_probability = np.mean(sample) / n_trials


Logikanya: kalau rata-rata sukses adalah 18 dari 30 percobaan, berarti peluang sukses = 18/30 = 0.6.

Uniform — uniform_params_estimation:

lower_bound = np.min(sample)
upper_bound = np.max(sample)


Paling simpel: batas bawah = angka terkecil, batas atas = angka terbesar!
