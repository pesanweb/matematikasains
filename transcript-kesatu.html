Kamu sudah bisa bikin angka acak uniform [0,1] dari Tugas 1.1. Nah, di Tugas 2.1 kamu bikin "mesin penerjemah" — fungsi yang bisa mengubah angka uniform [0,1] menjadi angka Gaussian (berdistribusi normal dengan μ dan σ tertentu).

  • σ = standar deviasi Gaussian yang diinginkan

Tugas 2.1 — uniform_inverse_cdf (Rumus Ajaib Inverse CDF)

erfinv adalah fungsi matematika khusus. probability itu bilangan antara 0 dan 1.

Tugas Segmen 2.2 — pdf_for_uniform (Hitung Peluang Uniform)

Tugas Segmen 2.3 — pdf_for_gaussian (Hitung Peluang Gaussian)



Rumus kurva lonceng (agak panjang tapi mudah):

1                -½ × ((val - miu) / sigma)²
pdf = ─────────── × e
       σ × √(2π)


Yang harus ditulis:

import numpy as np

pdf = (1 / (sigma  np.sqrt(2  np.pi)))  np.exp(-0.5  ((val - miu) / sigma)  2)


Diterjemahkan:
  • np.sqrt(2 np.pi) = akar dari (2 × 3.14159...)

  • np.exp(...) = e pangkat ... (e ≈ 2.718)

  • -0.5 ((val - miu) / sigma) 2 = minus setengah kali ((jarak dari rata-rata) dibagi sigma) dikuadratkan

Tugas Segmen 2.5 — Estimasi Parameter (Menebak miu, sigma, dll dari Data)



Kalau di dunia nyata, kita tidak tahu miu dan sigma sebenarnya. Kita harus menebak dari data yang ada. Ini caranya:

Gaussian — gaussian_params_estimation:

miu = np.mean(sample)
sigma = np.std(sample, ddof=1)  # ddof=1 untuk standar deviasi sampel


  • np.mean = rata-rata

  • np.std = standar deviasi (seberapa menyebar data)

  • ddof=1 = pakai rumus sampel (bukan populasi)


Binomial — binomial_params_estimation:

success_probability = np.mean(sample) / n_trials


Logikanya: kalau rata-rata sukses adalah 18 dari 30 percobaan, berarti peluang sukses = 18/30 = 0.6.

Uniform — uniform_params_estimation:

lower_bound = np.min(sample)
upper_bound = np.max(sample)


Paling simpel: batas bawah = angka terkecil, batas atas = angka terbesar!

Tugas Segmen 2.6 — training_params_estimation (Menebak Parameter dari Data Training)



Ini fungsi yang paling panjang. Tapi santai, logikanya sederhana: kita ulangi proses estimasi di atas untuk setiap jenis burung dan setiap ciri.

Yang harus ditulis:

# Loop untuk setiap jenis burung (0, 1, 2)
for breed in df['breed'].unique():
    # Ambil data hanya untuk jenis burung ini
    df_breed = df[df['breed'] == breed][features]
    
    # Hitung proporsi (prior probability)
    probs_dict[breed] = len(df[df['breed'] == breed]) / len(df)
    
    # Dictionary dalam untuk menyimpan parameter per fitur
    inner_dict = {}
    
    # Loop untuk setiap fitur (wingspan_cm, weight_g, dll)
    for feature in df_breed.columns:
        if feature == &quot;wingspan_cm&quot; or feature == &quot;weight_g&quot;:
            miu = df_breed[feature].mean()
            sigma = df_breed[feature].std(ddof=1)
            params = gaussian_params(miu=miu, sigma=sigma)
        elif feature == &quot;sing_days&quot;:
            n_trials = 30
            probability = df_breed[feature].mean() / n_trials
            params = binomial_params(n_trials=n_trials, probability=probability)
        elif feature == &quot;beak_head_ratio&quot;:
            lower_bound = df_breed[feature].min()
            upper_bound = df_breed[feature].max()
            params = uniform_params(lower_bound=lower_bound, upper_bound=upper_bound)
        
        inner_dict[feature] = params
    
    params_dict[breed] = inner_dict


> ⚠️ Perhatikan: Di sini kita tidak bisa pakai fungsi estimasi yang sudah dibuat sebelumnya (gaussian_params_estimation, dll) karena fungsi ini butuh akses langsung ke kolom DataFrame. Jadi kita hitung manual. ddof=1 penting untuk std()!

Tugas Segmen 2.7 — probability_given_class (Seberapa Mungkin Ciri Ini Milik Jenis X?)



Ini jantung Naive Bayes! Kita hitung: "Kalau burung ini jenis X, seberapa mungkin dia punya sayap segini, berat segini, dll?"

Caranya: kalikan semua peluang dari tiap ciri.

Yang harus ditulis:

for i, feature in enumerate(features):
    x = X[i]
    params = params_dict[breed][feature]
    
    if feature == &quot;wingspan_cm&quot; or feature == &quot;weight_g&quot;:
        probability_f = pdf_for_gaussian(x, params.miu, params.sigma)
    elif feature == &quot;sing_days&quot;:
        probability_f = pmf_for_binomial(int(x), params.n_trials, params.probability)
    elif feature == &quot;beak_head_ratio&quot;:
        probability_f = pdf_for_uniform(x, params.lower_bound, params.upper_bound)
    
    probability = probability_f


  • enumerate(features) = ngasih nomor urut dan nama fitur sekaligus

  • X[i] = nilai fitur ke-i

  • params_dict[breed][feature] = ambil parameter untuk jenis dan fitur tertentu

Tugas Segmen 2.8 — breed_bird_prediction (Tebak Jenis Burung!)



Langkah terakhir! Kita hitung skor untuk setiap jenis burung, terus pilih yang paling tinggi.

Rumus: skor = P(X | jenis) × P(jenis)

Yang harus ditulis:

posterior_breed_0 = probability_given_class(X, features, 0, params_dict)  probs_dict[0]
posterior_breed_1 = probability_given_class(X, features, 1, params_dict)  probs_dict[1]
posterior_breed_2 = probability_given_class(X, features, 2, params_dict) * probs_dict[2]

posterior = np.array([posterior_breed_0, posterior_breed_1, posterior_breed_2])
prediction = np.argmax(posterior)


  • np.argmax = cari indeks (posisi) nilai tertinggi

  • Kalau posterior_breed_0 paling besar → prediksi = 0

  • Kalau posterior_breed_1 paling besar → prediksi = 1

  • Kalau posterior_breed_2 paling besar → prediksi = 2

def generate_rand_uniform(lower_bound:float, upper_bound:float, num_samples:int):
    np.random.seed(15)  # JANGAN DIUBAH

    # MULAI KODE DI SINI
    array = np.random.uniform(lower_bound, upper_bound, num_samples)
    # AKHIRI KODE DI SINI

    return array
